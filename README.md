# Workshops

A curated portfolio of hands-on technical workshops designed, built, and delivered by Bruno Copa.

This repository collects practical workshop material across applied machine learning, data engineering, automation, and engineering enablement. Each workshop is organized so recruiters, collaborators, and participants can quickly understand the topic, learning outcomes, and available materials.

## Featured workshop: Energy Demand Forecasting

The first workshop in this collection is a notebook-based machine learning workshop focused on energy demand forecasting. It includes participant notebooks, solution notebooks, supporting project files, and delivery documentation.

The material demonstrates:

- End-to-end workshop and curriculum design
- Applied machine learning with time-series energy data
- Regression-based forecasting workflows
- Deep learning approaches with LSTMs
- Anomaly detection and autoencoder concepts
- Clear separation between participant material and instructor solutions
- Production-minded setup with Python project files and reproducible dependencies
